# Amaranth 10 신규 고객 가격확약서 생성 스킬

## 개요
Amaranth 10 최초 도입 고객 대상 가격확약서 xlsx 초안을 생성한다.
- **구축형(설치형)**: `a10_onpremise_template.xlsx` 템플릿 사용
- **클라우드형**: `a10_cloud_template.xlsx` 템플릿 사용

> ⚠️ 이 스킬은 xlsx 초안만 제공. PDF 변환은 사용자가 직접 수행.
> ⚠️ 할인율은 적용하지 않음 — 표준가로만 초안 제공.

## 견적 유형 판별

| 트리거 | 유형 | 템플릿 |
|-------|------|--------|
| "A10 구축형", "A10 설치형", "온프레미스" | 구축형 | a10_onpremise_template.xlsx |
| "A10 클라우드", "A10 Cloud" | 클라우드형 | a10_cloud_template.xlsx |
| 미지정 시 | 사용자에게 확인 | — |

## 템플릿 파일 위치
- 구축형: `templates/a10_onpremise_template.xlsx`
- 클라우드형: `templates/a10_cloud_template.xlsx`

> 파일이 없으면 사용자에게 업로드 요청

## 워크플로우

```
1. 견적 유형 판별 (구축형 / 클라우드형)
2. 사용자 입력값 확인 (고객사명, 유저수, 모듈, 교육비 유형)
3. 단가 참조 (douzone-pricelist 스킬 또는 표준가격표)
4. 템플릿을 /home/claude/ 에 복사 후 작업
5. 셀 스캔 후 항목명 기반으로 값 입력
6. xlsx 초안 제공 (PDF 변환 안 함)
```

### 작업 방법 (공통)
```python
import openpyxl
wb = openpyxl.load_workbook(template_path, data_only=False)
ws = wb[sheet_name]

# H열에서 항목명을 찾아 해당 행의 J열에 수량 입력
for row in range(1, ws.max_row + 1):
    item_name = ws.cell(row=row, column=8).value  # H열
    if item_name and '전문가' in str(item_name) and '유저' in str(item_name):
        ws.cell(row=row, column=10).value = user_count  # J열
```

## 모듈 선행조건 체크

```python
def check_prerequisites(modules):
    errors = []
    if '예산관리' in modules and '회계관리' not in modules:
        errors.append('예산관리는 회계관리 선도입 필요')
    if '자산관리' in modules and '예산관리' not in modules:
        errors.append('자산관리는 예산관리 선도입 필요 (비영리 전용)')
    if '외주관리' in modules and '생산관리' not in modules:
        errors.append('외주관리는 생산관리 선도입 필요')
    if '원가관리' in modules and '생산관리' not in modules:
        errors.append('원가관리는 생산관리 선도입 필요')
    if 'UC확장팩' in modules and not any(m in modules for m in ['회계관리','인사관리','영업관리','구매자재관리','생산관리']):
        errors.append('UC확장팩은 ERP 모듈 1개 이상 필요')
    return errors
```

## 참고: 견적 제안 시 안내사항
- 설치형은 유저 라이선스(일시납) + SSL(연납) + 방화벽 + 교육비 + 블루멤버십(연납)
