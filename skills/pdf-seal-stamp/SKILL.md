# PDF 인감(도장) 오버레이 스킬

## 개요
PDF 문서의 지정 페이지에 인감 이미지를 반투명 오버레이한다.
- 검은 배경 인감 → 빨간색만 추출 + 반투명 처리
- (인) 글자가 비쳐 보이도록 alpha 조절
- 위치(x, y), 크기, 투명도 파라미터 조정 가능

## 사용법

### 1단계: 인감 이미지 결정
- 사용자가 인감 이미지를 업로드한 경우 → 해당 이미지 사용
- 업로드하지 않은 경우 → 기본 더존비즈온 인감 사용: `{SKILL_DIR}/assets/douzone_seal.png`

### 2단계: 스크립트 실행

```bash
python3 {SKILL_DIR}/scripts/overlay_seal.py \
  "<원본PDF경로>" \
  "<출력PDF경로>" \
  --seal "<인감이미지경로>" \
  --page <페이지번호> \
  --x <X좌표> --y <Y좌표> \
  --size <인감크기> \
  --alpha <투명도>
```

### 3단계: 미리보기 확인
pdf2image로 해당 페이지를 렌더링하여 인감 위치를 확인한다.

```python
from pdf2image import convert_from_path
imgs = convert_from_path(output_path, first_page=page, last_page=page, dpi=200)
# 인감 영역만 크롭하여 확인
```

위치가 안 맞으면 x, y 조정 후 재실행.

### 4단계: present_files로 최종 파일 제공

## 파라미터 기본값 (더존 계약서 기준)

| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| page | 2 | 대상 페이지 (보통 계약서 2페이지 서명란) |
| x | 493 | X좌표 - 공급사(인) 위치 기준 |
| y | 100 | Y좌표 - 공급사(인) 위치 기준 |
| size | 58 | 인감 크기 (pt) |
| alpha | 180 | 투명도 (0=완전투명, 255=불투명). 180이면 뒤 글자가 비침 |

## 위치 조정 가이드
- A4 PDF 기준 페이지 크기: 약 595 x 842 pt
- **고객사(인)** 위치: 보통 x=240~270, y=100 부근 (왼쪽 테이블)
- **공급사(인)** 위치: 보통 x=490~510, y=100 부근 (오른쪽 테이블)
- 문서마다 다르므로 미리보기 후 조정 필수

## 주의사항
- 인감 이미지가 검은 배경인 경우 자동으로 빨간색 추출 + 배경 제거 처리됨
- 이미 흰 배경 + 빨간 인감인 경우에도 스크립트가 정상 동작함 (빨간 부분만 남김)
- 원본 PDF는 수정하지 않고 새 파일로 출력
