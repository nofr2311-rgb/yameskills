# WEHAGO 신규 고객 견적서 생성 스킬

## 개요
WEHAGO 최초 도입 고객 대상 견적서를 자동 생성한다.
- **신규(기업)**: Smart A 10 라이선스 + WEHAGO 사용료 + 그룹웨어 부가서비스
- **세무**: WEHAGO T (세무대리 전용)

## 견적 유형 판별

| 트리거 | 유형 | 시트 |
|-------|------|------|
| "WEHAGO 신규", "Smart A 견적", "기업 견적" | 신규(기업) | 신규(기업) |
| "WEHAGO T", "세무 견적", "세무대리" | 세무 | 세무 |

## 템플릿 파일
`templates/wehago_template.xlsx`

## 워크플로우

```
1. 견적 유형 판별 (신규기업 / 세무)
2. 사용자 입력값 확인 (고객사명, 유저수, 서비스 항목)
3. 템플릿 복사 → 값 채움
4. 불필요 시트 삭제 (PDF용)
5. LibreOffice PDF 변환
6. xlsx + pdf 함께 제공
```

## 필수 입력값

### 신규(기업)
- **고객사명** (필수)
- **Smart A 10 모듈 및 유저수**: 회계(M21), 인사급여(M22), 물류(M23), 법인조정(M24), 개인조정(M25), Smart A 2.0(M26)
- **WEHAGO 사용료 유저수**: Smart A 10 사용료(M31)
- **요금제 유형**: CLUB 또는 PRO (C32 셀 값으로 결정)
- **요금제 유저수**: M32 (요금제 기본), M33 (요금제 사용료)
- **납부기준**: 월납 또는 연납 (H29 셀 = "(월납)" 또는 "(연납)")
- **그룹웨어 부가서비스**: ONE AI(M41), 법정의무교육4종(M42), 법정의무교육5종(M43), 저장공간(M44), 경비청구법인(M45), 경비청구개인(M46), 전자결재(M47), 근태관리(M48), CRM(M49), PMS(M50)

### 세무
- **고객사명** (필수)
- **WEHAGO T 유저수**: P19 (기본 5유저 무상, 추가분만 과금)
- **타임머신 선택여부**: P20 (0 또는 1)
- **ONE AI**: P21 (개월수)
- **Smart A 2.0 수량**: P34
- **WEHAGO T 유저 추가**: P35 (추가 유저수)

## 엑셀 작업 규칙

1. 원본 템플릿을 `/home/claude/`로 복사 후 작업
2. `openpyxl data_only=False`로 열어 수식 보존
3. 병합셀 시작셀에만 write
4. M열(수량)만 채움 — O/S열(금액)은 수식 자동계산
5. 할인율(R열)은 0 유지 (표준가 초안)
6. 사용하지 않는 항목의 M열은 빈칸 또는 0

## PDF 생성 규칙

### 1단계: PDF용 임시 파일 생성
- 대상 시트만 남기고 나머지 시트 삭제 (신규(기업) 선택 시 세무/가격표 삭제, 세무 선택 시 신규(기업)/가격표 삭제)
- VLOOKUP 수식(U12, U13) → 직접 값으로 대체 (이영진 대리 정보)
  - U12: 010-4903-2311
  - U13: nofr2311@douzone.com

### 2단계: LibreOffice PDF 변환
```bash
libreoffice --headless --calc --convert-to pdf --outdir /home/claude/ /home/claude/wehago_pdf_temp.xlsx
```

### 3단계: 검증
```python
from pypdf import PdfReader
r = PdfReader('output.pdf')
for i, p in enumerate(r.pages):
    text = p.extract_text() or ''
    assert '#NAME' not in text
    assert '#REF' not in text
```

## 자동 체크리스트
1. 고객사명 입력 확인
2. 수량이 입력된 항목만 금액 발생 확인
3. 납부기준(월납/연납) 올바르게 설정
4. 요금제 유형(CLUB/PRO) 올바르게 설정
5. PDF에 #NAME?/#REF! 에러 없음
6. VLOOKUP 참조 시트(가격표) 삭제 전 직접값 대체 완료
