# Amaranth 10 갑지(가격제안서 요약) 생성

## 개요
Amaranth 10 견적서의 **갑지**(가격제안서 요약) 데이터를 읽어 **PDF + XLSX 양 포맷**으로 출력한다.
입력은 xlsx(갑지 시트 포함) 또는 pdf(갑지 단일 페이지) 모두 가능.

## v2 변경사항
- **비활성 유형 열 자동 숨김**: 총계/개별 항목 기준으로 데이터 없는 유형(Cloud/파킹/임대/자체) 열을 자동 숨김
- **빈 항목 행 자동 숨김**: 활성 열 기준 모든 값이 0/빈값인 항목 행 숨김 (EBP 엔진 섹션 헤더는 항상 표시)
- **기본 제안사 정보**: 더존비즈온 사업자번호, 주소, 대표이사 이강수·지용구
- **기본 영업담당**: 이영진 대리 / 010-4903-2311
- **견적번호**: 생략 (불필요)
- **제안 구성 섹션 PDF 파싱**: 서버 구매, 인프라 구성, ERP/UC 모듈, 사용자 구성 정상 추출

## 입출력
- **입력**: xlsx (갑지 시트 포함 견적서) 또는 pdf (갑지 페이지)
- **출력**: `<basename>.pdf` + `<basename>.xlsx` 동시 생성
- xlsx 출력은 원본 갑지 시트의 서식/병합/색상을 그대로 보존 (assets/galji_template.xlsx 템플릿 사용)

## 실행
```bash
python3 /path/to/a10-cover-sheet/scripts/generate_cover.py <input.xlsx|pdf> <output_basename>
```
예시:
```bash
python3 scripts/generate_cover.py 견적서_KBSN.xlsx /home/claude/갑지_KBSN
# → 갑지_KBSN.pdf + 갑지_KBSN.xlsx 생성

python3 scripts/generate_cover.py 갑지_원본.pdf /home/claude/갑지_복원
# → 갑지_복원.pdf + 갑지_복원.xlsx 생성
```

## 자동 숨김 로직
1. **열 숨김**: 총계행(일회납/월납/연납) + 개별 항목 기준, 4개 유형 중 데이터 있는 열만 표시
2. **행 숨김**: 활성 열 기준 모든 값이 0/빈값인 항목 행 숨김 (NEVER_HIDE_ROWS 예외: EBP 엔진 row 19)
3. PDF 출력은 숨김 적용된 xlsx를 LibreOffice로 렌더링하여 숨김이 반영됨

## 주의사항
- xlsx 입력 시 `data_only=True`로 수식 결과값 추출
- pdf 입력은 pdfplumber로 테이블 파싱 (복잡한 병합셀 구조 시 xlsx 입력 권장)
- xlsx 출력 시 `assets/galji_template.xlsx` 템플릿 사용 → 원본 서식 보존
- 금액: 천 단위 콤마, 0→"-", 문자열("자체구매")→그대로
- 최종 파일은 `/mnt/user-data/outputs/`에 복사 후 present_files로 제공
